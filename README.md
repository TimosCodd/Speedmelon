A effortless MelonLoader installer for **SpeedRunners 2: The King of Speed**.

**Speedmelon** is a tool designed to simplify the modding process for *SpeedRunners 2*. It automatically handles the installation of MelonLoader and ensures all necessary dependencies (unstripped DLLs) are correctly placed, so you can start modding immediately.

## Built With
* C# / .NET
